example_data_frame

data_frame_with_cleaner_names <- example_data_frame %>% 
  clean_names()

# Step 4: Compare these column names with the ones from above. Much cleaner!
data_frame_with_cleaner_names

2. In-class exercise: R code scratchpads

Question: Should I be copying/pasting/tweeking code directly in the console window?
Answer: You can, but it makes saving work difficult. Instead, create a ‚Äúscratchpad‚Äù:

  1. Create a new .R script: In RStudio Menu Bar -> File -> New File -> R Script.
  2. Save this as scratchpad.R
  3. Copy/paste/tweak any code in your scratchpad, not in console window. For example, for this exercise write: 1+1
  4. To ‚Äúrun‚Äù code in the console (in other words tell R to ‚Äúdo this‚Äù), you can do one of the following:
    1. Copy it from scratchpad, then paste it in console, then hit enter.
    2. Highlight the code you want to run, then click on ‚ÄúRun‚Äù
    3. Highlight the code you want to run, then press control+enter on Windows/Unix/Chromebook or ‚åò+enter on macOS.
  5. Save and close the file.
  6. In the Files panel, click on scratchpad.R to reopen your previously saved work.
